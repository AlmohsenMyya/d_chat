## 🚀 Overview

D-chat is a mobile chat application built using Flutter and Firebase.
It enables users to communicate instantly with a clean and responsive interface.

---

## 🧩 Features

* 🔐 User Authentication (Email & Password)
* 💬 Real-time Messaging
* 👥 User List & Private Chats
* 🟢 Online / Offline Status
* 🔔 Push Notifications
* ☁️ Cloud-based Data Storage

---

## 🛠️ Tech Stack

* Flutter
* Firebase

    * Firebase Authentication
    * Cloud Firestore
    * Firebase Cloud Messaging (FCM)

---

## 🏗️ Architecture

This project follows the **Client–Server model**:

* Flutter app → Client
* Firebase → Server

---

## 🎯 Project Goal

To build a clean, efficient, and real-time chat application using modern technologies.
